# Staff guide

## Signing in

- `/staff/login` — username + password.
- `/kiosk` — full-screen PIN pad for shared counter terminals.

Sessions expire automatically; you'll be returned to login with a notice.

## Your queue (`/staff`)

You see the queue for your assigned service only:

- **Now serving** card with a live "serving for" timer, customer name
  (when provided), notes, and priority/referred badges.
- **Next in Queue** indicator, then the full waiting list.
- **Call Next** — serves the current token and calls the next one.
- **⊘ No-show / Skip** — marks the called token expired.
- **Refer to another counter →** — e.g. OPD doctor sends a patient to the
  Eye Specialist. The token keeps its number, joins the destination queue
  with priority, and never auto-expires mid-transfer.
- Add a note to the current token (visible to admin and the display board).
- A banner warns when priority customers are waiting at other counters.

## Workspace tools

Same as admins: ✦ AI Assistant (bottom-left), 💬 docked messaging tray
(bottom-right — chat 1:1 or in groups with the whole team), 🔔
notifications, and 📁 shared files. All panels close automatically when
you switch screens.

## Presence

While signed in you appear **online** to admins (green dot); closing the
tab marks you offline automatically.
