# QueueLess — v1.6.0

**Codename: Quasar — Forge**

## 📖 Overview

Quasar rebuilds team messaging as a **LinkedIn-style docked tray** — a
bottom-anchored bar that slides a chat panel up over any screen, remembers
its state across navigation, and shows a live unread badge even while
collapsed.

Building on the messaging core shipped in Zenith, this release focuses on
the *surface*: chat should feel like a lightweight app-within-the-app, not
a floating bubble competing with page content.

## 🎯 Objectives

- Replace the floating chat bubble with a bottom-docked message tray.
- Persist the tray's open/closed state across navigation.
- Keep the unread badge live without polling.

## ✨ Added

### 🪟 Docked message tray

- Bottom-anchored bar with a smooth, hardware-accelerated slide-up panel.
- **Persistent state** via `localStorage` — the tray reopens exactly where
  you left it after navigating.
- **Live unread badge** on the collapsed bar, driven by the content-free
  `messageSignals` RTDB node (no polling).

## 🔄 Changed

- Messaging moved to the bottom-right dock; the ✦ AI assistant button
  relocated to bottom-left so the two never collide.
- Footer given clearance so floating docks never cover the status bar.

## 🐛 Fixed

- Collapsed tray no longer overlaps the footer version / location text.

## 🏛️ Architecture Progress

The tray keeps the Zenith security model untouched — content via the JWT
API, real-time via signal nodes — proving the UI can be rebuilt freely on
top of the same transport.

Known limitations at this release:

- Attachments remain bounded at 256 kB inline (Spark plan).
